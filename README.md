### Installation

1. Clonez le projet
   ```sh
   git clone https://github.com/RomainC17/CroquettesClicker.git
   ```
2. Installez les packages NPM 
   ```sh
   npm install
   ```
3. Lancez le projet
   ```sh
   npm run start
   ```
